# 중간점검 발표용 5장 슬라이드 문구

## 슬라이드 1. 프로젝트 한 줄 설명

### PPT 문구

공공조달 수요 데이터를 활용한 예비창업자 사업 아이템·입지 추천 보조 서비스

- 입찰공고를 지역별·품목별 공공수요 신호로 해석
- 예비창업자와 창업상담가를 위한 참고 근거 제공
- 창업 성공 예측이 아닌 의사결정 보조 도구

### 발표 멘트

이 프로젝트는 창업 성공을 맞히는 서비스가 아닙니다. 공공조달 입찰공고에서 지역별·품목별 수요 신호를 찾아, 창업상담 과정에서 참고할 수 있는 근거 자료를 만드는 것이 목표입니다.

## 슬라이드 2. 문제의식

### PPT 문구

악성재고는 수요를 잘못 읽은 결과

- 물류창고 10년 현장 경험
- 재고 불균형은 보관비·처리비·손실로 연결
- 창업도 지역·품목 미스매치 시 재고 부담 발생
- 공공데이터로 수요 판단 근거를 만들고자 함

### 발표 멘트

저는 물류창고에서 일하며 악성재고가 단순히 남은 물건이 아니라 수요와 공급이 어긋난 결과라는 점을 체감했습니다. 이 문제의식을 창업 전 아이템과 입지를 검토하는 과정에도 적용해보고자 했습니다.

## 슬라이드 3. 데이터와 처리 흐름

### PPT 문구

조달 API → 정제 → 지역 x 품목군 매트릭스 → 점수화

- 조달청 입찰공고 API 수집
- 공고명, 수요기관, 금액, 공고일 정제
- 서울 자치구명 추출
- 공고명 기반 품목군 분류
- 공고 수, 금액 규모, 최근성 기반 opportunity_score 계산

### 발표 멘트

현재는 서울 주요 자치구를 대상으로 샘플을 만들었습니다. 입찰공고를 정제한 뒤 자치구와 품목군을 붙이고, 공고 수·금액·최근성을 결합해 공공수요 참고 점수를 계산했습니다.

## 슬라이드 4. 현재 결과물

### PPT 문구

결과물 3종

- 자치구별 추천 품목 TOP3
- 품목군별 추천 자치구
- 지역 x 품목군 히트맵
- 자동 요약 리포트

샘플 해석:

- 종로구: 도서/콘텐츠 관련 공고 다건 포착
- 강남구: IT/소프트웨어 및 환경개선 관련 공고 확인
- 마포구: 창업/경영지원 관련 공고 확인

### 발표 멘트

아직은 MVP 샘플이지만, 결과표와 히트맵을 통해 어떤 지역에서 어떤 품목군이 공공수요로 포착되는지 확인할 수 있습니다. 이 결과는 상담가가 다른 상권 정보와 함께 검토할 수 있는 보조 자료로 볼 수 있습니다.

## 슬라이드 5. 한계와 다음 단계

### PPT 문구

현재 한계

- 서울 일부 자치구 중심 샘플
- 일부 관심 자치구는 공고 수 부족
- 품목군 분류는 키워드 기반 초기 버전
- 인구/세대 및 낙찰정보는 아직 미결합
- opportunity_score는 공공수요 참고 지표

다음 단계

- 주민등록 인구 및 세대현황 결합
- 인구 대비 공공수요 지표 생성
- 낙찰정보 결합
- 품목군 키워드 사전 개선
- 지역 특성 기반 클러스터링 확장

### 발표 멘트

현재 결과는 완성형 모델이 아니라 방향을 검증하기 위한 MVP입니다. 다음 단계에서는 인구와 세대수, 낙찰정보를 결합해 지역 규모를 보정하고, 이후에는 지역 특성 기반 클러스터링으로 확장할 계획입니다.
