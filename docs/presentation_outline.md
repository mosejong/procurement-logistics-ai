# 발표 구성안

## 1. 문제 제기

- 고물가 시대에 창업을 고민하는 예비창업자가 늘고 있습니다.
- 하지만 어떤 지역에서 어떤 품목으로 시작할지 판단할 근거를 찾기 어렵습니다.
- 현장 경험상 악성재고는 수요를 잘못 읽은 결과이며, 창업에서도 수요와 맞지 않는 품목 선택은 재고 부담과 손실로 이어질 수 있습니다.

## 2. 개인적 문제의식

- 물류창고에서 10년간 근무하며 악성재고와 재고 불균형 문제를 경험했습니다.
- 재고회전율이 낮아지면 보관비와 처리비가 증가하고 영업손실로 이어집니다.
- 이 문제의 본질은 수요와 공급의 불일치라고 보았습니다.

## 3. 프로젝트 방향

- 이미 발생한 악성재고를 처리하는 사후 대응이 아니라, 창업 전 단계에서 수요 신호를 참고할 수 있도록 돕는 사전 예방형 접근입니다.
- 공공조달 입찰공고를 지역별·품목별 공공수요 신호로 해석합니다.
- 예비창업자와 창업상담가가 사업 아이템과 입지를 검토할 때 참고할 수 있는 데이터 근거를 제공합니다.

## 4. 데이터와 분석 방법

- 조달청 입찰공고 API를 수집합니다.
- 공고명, 수요기관, 지역, 공고일, 추정금액을 정제합니다.
- 서울 자치구명과 품목군을 추출합니다.
- 지역 x 품목군 매트릭스를 만듭니다.
- 공고 수, 금액 규모, 최근성을 기준으로 `opportunity_score`를 계산합니다.

## 5. 결과물

- 자치구별 추천 품목 TOP N
- 품목군별 추천 자치구 TOP N
- 지역 x 품목군 히트맵
- 자동 생성 요약 리포트
- Streamlit 중간점검 화면

## 6. 한계와 고도화

- 조달 데이터는 공공기관 수요를 보여주므로 민간 소비 수요 전체를 대변하지는 않습니다.
- 창업 성공을 예측하는 모델이 아니라 판단 근거를 제공하는 보조 도구입니다.
- 향후 인구/세대, 낙찰정보, 성·연령별 인구, 상권 데이터를 결합할 예정입니다.
- 지역 특성 기반 클러스터링 모델로 확장할 수 있습니다.
