# 프로젝트 아이디어 초안

## 프로젝트명

공공조달 수요 기반 사업 아이템·입지 추천 보조 서비스

## 한 줄 설명

공공조달 입찰공고를 지역별·품목별 공공수요 신호로 분석해, 예비창업자와 창업상담가가 사업 아이템과 입지를 검토할 때 참고할 수 있는 근거 자료를 제공합니다.

## 개발 동기

물류창고에서 10년간 근무하며 가장 크게 체감한 문제 중 하나는 악성재고와 재고 불균형이었습니다. 수요를 잘못 예측하거나 지역별 수요 차이를 반영하지 못하면 물건이 특정 창고에 쌓이고, 보관비와 처리비가 발생해 손실로 이어집니다.

창업도 마찬가지입니다. 수요가 충분하지 않은 지역에 맞지 않는 품목으로 진입하면 재고 부담과 운영 손실이 커질 수 있습니다. 따라서 창업 전 단계에서 지역별 수요 신호를 참고할 수 있다면 더 신중한 의사결정에 도움이 될 수 있습니다.

본 프로젝트는 조달청 입찰 공고 데이터를 활용해 지역별·품목별 공공수요를 정리하고, 창업상담 과정에서 참고할 수 있는 데이터 기반 근거를 제공하는 것을 목표로 합니다.

## 문제 정의

고물가 시대에 생계와 미래를 위해 창업을 고민하는 사람이 늘고 있습니다. 그러나 많은 예비창업자는 어떤 지역에서 어떤 품목으로 시작해야 할지 감이나 주변 경험에 의존하는 경우가 많습니다.

상권, 임대료, 유동인구 같은 민간 데이터도 중요하지만, 공공기관이 실제로 어떤 품목을 반복 구매하는지 보여주는 공공조달 데이터 역시 지역 수요를 읽는 하나의 근거가 될 수 있습니다.

## 제안 솔루션

1. 조달청 입찰 공고 데이터를 수집합니다.
2. 공고명, 기관명, 지역, 공고일, 추정금액을 표준화합니다.
3. 공고명과 품목 키워드를 기반으로 품목군을 분류합니다.
4. 지역·품목 매트릭스를 만듭니다.
5. 공고 수, 금액 규모, 최근성을 결합해 `opportunity_score`를 계산합니다.
6. 지역을 입력하면 추천 품목을, 품목을 입력하면 추천 지역을 제공합니다.
7. 추천 이유를 공고 수, 금액, 최근성 지표와 함께 설명합니다.

## 현재 MVP 범위

- 서울 주요 자치구 샘플 분석
- 조달청 입찰공고 API 기반 수집
- 규칙 기반 품목군 분류
- 지표 기반 `opportunity_score` 산출
- Streamlit 중간점검 화면 제공

## 향후 확장

- 주민등록 인구 및 세대현황 API 결합
- 낙찰정보 API 결합
- 성·연령별 인구 구조 반영
- 지역 특성 기반 클러스터링
- 상권/매출/임대료 데이터 결합

## 기대 효과

- 예비창업자의 막연한 판단을 줄이고 데이터 기반 검토를 돕습니다.
- 창업상담가가 상담 과정에서 참고할 수 있는 보조 자료를 제공합니다.
- 공공조달 데이터를 사업기회 탐색에 활용하는 사례를 제시합니다.
- 수요와 맞지 않는 품목 선택으로 인한 재고 부담 가능성을 사전에 줄이는 데 기여합니다.
