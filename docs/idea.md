# 프로젝트 아이디어 초안

## 프로젝트명
공공조달 수요 기반 물품 배치 및 이동 최적화 시스템

## 문제 정의
공공기관의 물품 조달은 지역별 수요 차이와 사용 패턴 차이를 충분히 반영하지 못하면
과잉 조달 또는 부족 현상이 발생할 수 있다.

이 프로젝트는 조달청 공공데이터와 타 공공데이터를 결합해
지역별 수요 특성을 분석하고 물품 배치 및 이동 효율화를 위한 인사이트를 제공하는 것을 목표로 한다.

## 활용 데이터
- 조달청 입찰공고 / 낙찰 / 계약 관련 데이터
- 지역별 인구 데이터
- 행정구역 데이터
- 필요 시 공공시설 분포 데이터

## 기대 효과
- 공공 물품 운영 효율 향상
- 불필요한 비용 감소
- 지역 특성을 반영한 배치 전략 도출
- 공공데이터와 AI 결합 사례 제시
