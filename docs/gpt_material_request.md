# GPT에게 요청할 자료 초안

다음 단계에서 GPT나 다른 자료 조사 도구에 요청하면 좋은 내용이다.

## 1. 공모전 기획서 문장 다듬기

요청 예시:

```text
공공조달 데이터를 활용해 예비창업자와 창업상담가에게 지역별·품목별 공공수요 근거를 제공하는 프로젝트를 준비 중이야. 물류창고 10년 경험에서 악성재고와 수요 불일치 문제를 느꼈고, 이를 창업 전 의사결정 보조로 확장하려고 해. 이 내용을 공모전 기획서의 개발 배경, 문제 정의, 기대 효과 문장으로 정리해줘.
```

## 2. 필요한 추가 데이터 목록

요청 예시:

```text
공공조달 데이터 기반 창업 아이템·입지 추천 프로젝트를 고도화하려면 어떤 공공데이터를 추가로 결합하면 좋을까? 유동인구, 인구구조, 임대료, 업종 매출, 경쟁 점포, 상권 특성 관점에서 데이터 후보와 활용 이유를 정리해줘.
```

## 3. 품목 분류 체계

요청 예시:

```text
조달 입찰공고명을 기반으로 창업 아이템 추천에 쓸 품목군을 만들려고 해. 사무용품, IT장비, 위생용품, 가구, 교육기자재, 시설관리, 행사홍보 등으로 분류할 수 있는 키워드 사전을 만들어줘.
```

## 4. 발표 스토리라인

요청 예시:

```text
물류창고 현장 경험에서 출발한 악성재고 문제를 공공조달 데이터 기반 창업상담 보조 서비스로 연결하는 발표 스토리라인을 만들어줘. 심사위원이 납득하기 쉽게 문제-데이터-분석-결과-한계-고도화 순서로 정리해줘.
```

## 5. 한계와 방어 논리

요청 예시:

```text
조달 데이터는 공공기관 수요라서 민간 창업 수요 전체를 대변하기 어렵다는 한계가 있어. 이 프로젝트를 창업 성공 예측이 아니라 창업상담 보조 데이터로 설명할 때 쓸 수 있는 방어 논리와 표현을 정리해줘.
```
