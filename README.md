# Procurement Logistics AI

공공조달데이터와 AI를 활용해 지역별 수요를 분석하고, 물품 배치 및 이동 효율화를 탐색하는 프로젝트입니다.

## 📌 About
- 조달청 공공데이터 활용
- 공공 물품 수요 및 지역 특성 분석
- 물류/배치 최적화 아이디어 도출
- 공모전 및 포트폴리오용 프로젝트

## 🛠 Tech Stack
- Python
- Pandas
- Requests
- Scikit-learn
- Matplotlib

## 📂 Structure
- `src/api` : 조달청 API 연동
- `src/collect` : 데이터 수집
- `src/preprocess` : 데이터 정제
- `src/analysis` : 분석 및 추천
- `outputs` : 결과물 저장
- `docs` : 아이디어/기획 문서
